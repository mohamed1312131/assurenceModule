# SIRH visual reference

The target UI theme is enterprise admin dashboard / internal tool.

Important visual traits:

- Dark left navigation.
- Grouped modules in sidebar.
- Active menu item state.
- Light main canvas.
- Rounded main shell.
- Breadcrumb topbar.
- Page title + subtitle.
- Search and advanced filter controls.
- Action buttons on the right.
- Cards for summaries and employees.
- Tables for operational data.
- Card/list switch where useful.
- Pagination and customizable columns.
- Calm, dense, professional layout.

Avoid:

- Landing page sections.
- Hero banners.
- Big decorative illustrations.
- Random colors.
- Overly playful UI.
- Too much whitespace.
- Unnecessary animations.
